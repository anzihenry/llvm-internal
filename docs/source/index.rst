Welcome to LLVM!
============================

本书主要主要探讨两个方面：

1. LLVM编译工具链上不同底层组件的作用以及各个组件的实现原理，同时会对编译器的设计进行深入讨论。
2. 如何对LLVM功能进行扩展，以及如何将扩展的功能应用到编译器中。

.. note::

   This project is under active development.
